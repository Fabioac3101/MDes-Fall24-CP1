Interactive p5.js Sketch

Instructions:

Click and drag mouse to draw

Press 'P' to create pattern

Press 'D' to delete pattern

The hardest part about creating a sketch in p5.js is knowing what to start with. Looking for some inspiration, I found myself browsing through the examples listed on p5.js examples. The most interesting interaction I saw was a sketch called "Drawing Lines." I had no clue that one could draw in a program like this. I decided to use this as the starting point.

In the sketch example, the strangest function I saw was "mouseDragged." This started a long journey in trying to get this to work. I initially thought that this was enough to start the drawing but realized that this only starts the drawing program when the mouse is clicked. It quickly snowballed into figuring out how to keep track and record the position of the mouse in order to place a circle pattern (through ellipses), then how to store the positioning so that it can be patterned.

After being able to pattern the drawing, I thought it would be nice to make it so that the user can add to the pattern and delete it to start over. The first try ended up working with an if statement reading which key was pressed and either allowing drawing or deleting it.
